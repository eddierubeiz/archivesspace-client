Archivesspace Client
====================

Interact with ArchivesSpace via its RESTful API.

Installation
------------

Add this line to your application's Gemfile:

```
gem 'archivesspace-client'
```

And then execute:

```
bundle
```

Or install it yourself as:

```
gem install archivesspace-client
```
